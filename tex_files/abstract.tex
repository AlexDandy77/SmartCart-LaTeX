The project titled \textbf{"SmartCart - Track Your Shopping with IoT"} was developed by students Alexei Pavlovschii, Alexandru Rudoi, Artur Tugui, Nicolae Marga, and Ion Vornicescu from the Technical University of Moldova.

This project comprises 5 chapters: Problem Framing, Domain Analysis, Solution Proposal, System Design and Practical Implementation, as well as Introduction, Conclusions and Bibliography.


\textbf{Keywords: } IoT, SmartCart, Shopping, Tracking.
